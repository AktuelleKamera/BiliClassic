# BiliClassic for webOS

一个面向 HP TouchPad (webOS 3.0) 设备的 Bilibili 客户端，让被遗忘在抽屉里的 webOS 设备重新获得观看 Bilibili 视频的能力。

## 已实现功能 (0.3.0)

- 支持 HP TouchPad (webOS 3.0) 设备
- 基于 Enyo 框架开发
- 黑色主题 / 粉色主题切换
- 历史记录
- 推荐视频
- 收藏管理
- 观看视频
- 搜索视频
- 检查更新
- 手动 Cookie 登录

---

## 计划中功能

- 查看评论区
- 弹幕支持
- 扫码登录

---

## 技术路线

- 基于 Enyo 2.5.1 / webOS 3.0 原生框架
- JavaScript 实现
- 使用 HTML5 `<video>` 标签播放 H.264 视频
- 参考 Bilibili 官方 API 文档实现数据获取

---

## 兼容性说明

本项目的目标设备是 HP TouchPad (webOS 3.0)。

### 配置
- 系统: webOS 3.0
- 处理器: 1.2GHz ARMv7双核
- 物理内存: 1GB

---

## 本项目是如何出现的？

开发者做了 Android 版之后发现，既然 webOS 看起来很好开发，自己的TouchPad也在吃灰，那为什么不顺便做个原生客户端呢？于是就有了这个新版本。目前基本只能看的说……
